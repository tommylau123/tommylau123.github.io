Step-by-Step: Check for Windows Updates

This quick check ensures your system has the latest security patches.

Step 1: Open Settings

Click the **Start Button** or **Search Icon** and type "Settings," then click the app icon to open the main settings window.

Step 2: Navigate to Windows Update

In the Settings window, click on **Windows Update** in the left-hand navigation pane.

Step 3: Checking for Updates

On the Windows Update screen, click the **Check for updates** button to begin the scan. If updates are found, they will start downloading automatically.
