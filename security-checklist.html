Step-by-Step: Check Windows Firewall

Follow these steps to verify that your Windows Firewall is active:

Step 1: Open Windows Security

Go to Settings → Privacy & Security → Windows Security

Step 2: Open Firewall & Network Protection

Click Firewall & network protection to view network profiles.

Step 3: Verify Network Profiles

Ensure Domain, Private, and Public networks all show Active / On.

Step-by-Step: Check for Windows Updates

This check ensures your system has the latest security patches.

Step 1: Open Settings

Click the Start Button or Search Icon, type Settings, and open the app.

Step 2: Navigate to Windows Update

Select Windows Update from the left-hand menu.

Step 3: Check for Updates

Click Check for updates. Updates will download automatically if available.

Check User Accounts Using Command Prompt (CMD)

This tutorial helps identify local user accounts and administrator privileges.

Step 1: Open Command Prompt

Open the Start Menu, type cmd, and click Command Prompt.

Step 2: List Local User Accounts

net user

Step 3: Check Administrator Accounts

net localgroup administrators
